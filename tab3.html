Side Projects

  • DubDubGrub Project

    This project aims to fulfill the usual take home project's requirements. It uses a programmatic custom UI, network calls, supports dark mode, persistence of data, inheritence and other iOS APIs, but avoids the usage of 3rd party libraries entirely.

  • BitCoin Tracker

    This is a simple BitCoin currency tracker that performs network calls and saves the last successful call. To spice things up a gesture for switching between light and dark mode with a neat little transition effect was implemented.
