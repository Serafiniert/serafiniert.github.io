Side Projects

  • AI Chat App

    As part of an advanced SwiftUI architecture course, I built an AI Chat App where users can generate custom avatars and chat with them in real time. Starting from vanilla SwiftUI, I progressively refactored the codebase through MVC, MVVM, and VIPER/RIBs architectures, gaining a deep understanding of scalable, enterprise-ready app design.

    Key areas and features I implemented:

    • AI-powered avatar generation and real-time chat
    • Sign in with Apple and anonymous authentication via Firebase
    • In-app purchases and paywall using StoreKit & RevenueCat
    • Local persistence with SwiftData
    • Fully modular codebase with reusable Swift Packages
    • Architecture refactor through MVC, MVVM, VIPER, and RIBs
    • 100% testable codebase with UI and unit tests at every layer
    • CI/CD pipeline with Xcode Cloud and Firebase Cloud Functions
    • Dependency Injection and Protocol-Oriented Programming across every layer
    • A/B testing via Firebase Remote Config

  • Vision Cam - Object Classification & Video Filters

    Vision Cam processes live video from the device camera and supports two object recognition models: one specialized in breakfast items (including bounding boxes) and another that detects a wide range of objects, returning the highest-confidence results. The app supports both front and back cameras, can mirror the image, and offers a variety of image filters. This project was built to explore live video stream handling, image processing, and visual object recognition in Swift.
